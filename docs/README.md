# Agent Operating System Documentation

Welcome to the comprehensive documentation for the Agent Operating System (AOS) Kernel.

The AOS Kernel is Foundry Agent Service native — all agent operations (CRUD, threads, runs) are performed via the azure-ai-projects/azure-ai-agents SDK.
